Loup-Garou

Description

Ce projet est une sorte de Wikipédia pour le jeu de société Loup-Garou de Thiercelieux. Ce jeu répond à un besoin personnel pour pouvoir jouer au Loup-Garou.
En effet, n'ayant pas le jeu physique, il me fallait un moyen pour y jouer.

Concept

Le concept de cette application web est de regrouper les principales cartes du jeu pour y avoir accès sur un téléphone.
Il inclut également les descriptions et quelques conseils pour jouer chaque rôle.
Pour cela, il est facilement possible de déployer ce projet sur un serveur.

Réalisation

Ce projet a été réalisé en HTML et CSS en 1 journée.
